## Skills

### AudioMind — AI Podcast Studio
[![GitHub](https://img.shields.io/badge/repo-skill--audiomind-blue?logo=github)](https://github.com/wells1137/skill-audiomind)

Turn any idea into a finished podcast in one command. Orchestrates ElevenLabs TTS (29+ voices), AI background music, and server-side audio mixing.

**Install:**
```bash
git clone https://github.com/wells1137/skill-audiomind.git ~/.manus/skills/audiomind
```

### Imagen — AI Image Generation
[![GitHub](https://img.shields.io/badge/repo-skill--imagen-blue?logo=github)](https://github.com/wells1137/skill-imagen)

Generate images using Google Gemini — UI mockups, illustrations, icons, concept art, and any visual content from text descriptions.

**Install:**
```bash
git clone https://github.com/wells1137/skill-imagen.git ~/.manus/skills/imagen
```
